# Generated Rust API

Regexped generates Rust FFI stubs that call into compiled WASM regex modules. This document explains how to include the stubs in your project and how to use the generated functions.

## Including stubs in your project

Each stub file is a plain Rust source file intended to be pulled in with `include!`. Because all regex WASM modules export functions with the same names (`match`, `find`, `groups`), each stub must live in its own `mod` block to avoid FFI name collisions:

```rust
// When you have multiple patterns, wrap each in a module:
mod github  { include!("github_stub.rs"); }
mod url_pat { include!("url_stub.rs");    }

// If you have only one pattern you can include it directly:
include!("stub.rs");
```

Stub files are written to `<out-dir>/<stub_file>` (include any subdirectory in `stub_file`, e.g. `src/stubs.rs`) and are typically added to `.gitignore` since they are regenerated by `regexped generate`.

## Generated functions by config field

### `match_func` ‚Äî anchored match

```rust
pub fn <match_func>(input: &[u8]) -> Option<usize>
```

Tries to match the pattern at position 0 of `input`. Returns the **end position** (exclusive) of the match, or `None` if no match.

```rust
match url_match(input) {
    Some(end) => println!("matched {} bytes", end),
    None      => println!("no match"),
}
```

---

### `find_func` ‚Äî non-anchored find iterator

```rust
pub fn <find_func>(input: &[u8]) -> FindIter<'_>
// FindIter implements Iterator<Item = (usize, usize)>
```

Returns an iterator that yields all non-overlapping matches as absolute `(start, end)` byte pairs. After a zero-length match the iterator advances by one byte to avoid infinite loops.

```rust
// All matches:
for (start, end) in find_token(input) {
    println!("match at {}..{}: {:?}", start, end, &input[start..end]);
}

// First match only:
if let Some((start, end)) = find_token(input).next() {
    println!("first match: {:?}", &input[start..end]);
}
```

### `groups_func` ‚Äî anchored capture groups iterator

```rust
pub fn <groups_func>(input: &[u8]) -> GroupsIter<'_>
// GroupsIter implements Iterator<Item = Vec<Option<(usize, usize)>>>
```

Returns an iterator that yields all non-overlapping matches. Each item is a `Vec` of capture group positions (absolute offsets). Index 0 is the full match; subsequent indices are capture groups in order. A group that did not participate is `None`.

```rust
// All matches:
for groups in parse_groups(input) {
    if let Some(Some((s, e))) = groups.get(1) {
        println!("group 1: {:?}", &input[*s..*e]);
    }
}

// First match only:
if let Some(groups) = parse_groups(input).next() {
    if let Some(Some((s, e))) = groups.get(1) {
        println!("group 1: {:?}", &input[s..e]);
    }
}
```

### `named_groups_func` ‚Äî named capture groups iterator

```rust
pub fn <named_groups_func>(input: &[u8]) -> NamedGroupsIter<'_>
// NamedGroupsIter implements Iterator<Item = HashMap<&'static str, (usize, usize)>>
```

Returns an iterator that yields all non-overlapping matches. Each item is a `HashMap` of name ‚Üí `(start, end)` for groups that participated in the match. Keys are `&'static str` ‚Äî group names are hardcoded at stub-generation time.

```rust
// All matches:
for parts in parse_url(text) {
    if let Some((s, e)) = parts.get("host") {
        println!("host: {}", std::str::from_utf8(&text[*s..*e]).unwrap());
    }
}

// First match only:
if let Some(parts) = parse_url(text).next() {
    if let Some((s, e)) = parts.get("host") {
        println!("host: {}", std::str::from_utf8(&text[*s..*e]).unwrap());
    }
}
```

## Notes

- Stub functions are safe Rust wrappers; the `unsafe` FFI call is hidden inside the stub.
- The WASM module must be loaded before calling any stub function. In `wasip1` targets, all imports are resolved at link time by the runtime.
- When multiple patterns are included in the same file without `mod` wrappers, their FFI declarations will conflict. Always use `mod` blocks when including more than one stub.
